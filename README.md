# Coursefolio

中国古代文学史在线课程纪念复活项目。

这个项目用于复活一个早年基于 WordPress 搭建的课程网站。原站代码和媒体资源已经丢失，只保留下 WordPress XML、SQL 备份和少量截图。因此本项目不追求完整还原旧 WordPress 系统，而是用更简单、可维护的静态站方式，尽可能恢复它的结构、文字内容和视觉记忆。

这是一个纪念项目，纪念 Jerry 的第一产品。

## 当前目标

- 恢复“中国古代文学史在线课程”的主要页面结构。
- 从 WordPress XML 中提取可用的课程文字、课时、栏目和学生实践内容。
- 参考旧截图重建页头、导航、内容容器、教师介绍页和页脚。
- 在媒体资源缺失的前提下，优先保证文字和页面结构可访问。
- 使用公开网页补充授课教师头像与简介来源。

## 技术选择

项目采用 Astro + TypeScript + 纯 CSS。

选择这个技术栈的原因很简单：这个站点的本质是纪念性展示和内容恢复，不需要后台、数据库、登录系统、WordPress 插件生态或复杂 LMS 功能。静态站可以把维护成本降到最低，也方便长期保存、部署和迁移。

## 内容来源

- 旧站备份：WordPress XML 与 SQL 备份。
- 旧站参考：用户保留的页面截图。
- 教师资料：广州南方学院文学与传媒学院公开页面。
- Logo：用户提供的 `logo1.png`。

原 WordPress 备份中包含页面、文章、课程和课时数据，但不包含真实媒体文件本体。旧站媒体链接也已不可用，所以本项目不把媒体完整恢复作为目标。

## 已恢复页面

- 首页
- 课程概述
- 授课教师简介
- 授课目标
- 课程大纲
- 参考资料
- 学生课程实践
- 中国古代文学史在线学习课程

- 教师详情页

## 本地运行

```powershell
npm install
npm run dev
```

默认本地地址：

```text
http://127.0.0.1:4321/
```

构建静态站：

```powershell
npm run build
```

## 项目结构

```text
src/
  data/        课程与教师数据
  layouts/     全站布局
  pages/       页面路由
  styles/      全局样式
public/
  images/      Logo 与教师头像
scripts/
  extract-wp-content.mjs  WordPress XML 内容提取脚本
```

## 设计原则

- KISS：能静态解决的问题，不引入复杂系统。
- 纪念优先：保留旧站气质，而不是重做一个现代产品。
- 事实优先：备份里没有的内容不伪造。
- 可维护：让未来只需要普通前端知识就能继续修改。
